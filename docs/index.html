Documentation CLI

Guide d'utilisation du Client CLI

Découvrez toutes les commandes disponibles pour interagir avec le serveur.

Indexation & Recherche

-s <dossier>

Démarre le processus d'indexation sur un dossier donné. Le serveur va scanner tous les fichiers textes et PDF, extraire leur contenu et peupler l'index inversé.

> -s src/documents
Dossier indexé avec succès (142 fichiers).
-r <mots>

Effectue une recherche intelligente. Le système utilise l'algorithme TF-IDF pour vous retourner les documents les plus pertinents classés par score. Vous pouvez utiliser les opérateurs logiques.

> -r architecture AND réseau
Document: src/cours.pdf | Score TF-IDF: 0.8452
Document: src/notes.txt | Score TF-IDF: 0.1205

Gestion des Fichiers

-dl <fichier>

Télécharge un fichier depuis le serveur vers votre machine locale (dans le dossier downloads/). Le transfert binaire garantit qu'il n'y a aucune corruption.

> -dl src/cours.pdf
-m <fichier>

Affiche les métadonnées internes générées par le moteur pour un document spécifique (Poids, date d'indexation, taille, etc).

> -m src/notes.txt
-m -rn <old> <new>

Renomme un fichier sur le serveur. Le moteur mettra automatiquement à jour l'index inversé et l'historique de journalisation.

> -m -rn vieux.txt nouveau.txt
-m -rm <fichier>

Supprime définitivement un fichier du serveur et purge ses données de l'index de recherche.

> -m -rm src/brouillon.txt

Outils Avancés

Le moteur de recherche embarque des modules complémentaires pour l'analyse et la maintenance de votre base documentaire.

-exif <image> : Extraction de Métadonnées Images

Utilise l'outil système exiv2 pour lire et afficher toutes les données cachées dans une image (Appareil photo, GPS, Date de création).

> -exif images/vacances.jpg
-d <fich1> <fich2> : Détection de Doublons

Compare mathématiquement deux fichiers pour déterminer s'ils sont strictement identiques en termes de contenu, peu importe leur nom.

> -d src/A.txt src/copie_de_A.txt
-sw -add / -rm : Gestion des Stop Words

Ajoute ou retire des "mots vides" (le, la, et, ou...) à la liste d'exclusion pour améliorer la pertinence du score TF-IDF.

> -sw -add donc,cependant
